Инструкция по использованию карты

  • 🔍 Verwenden Sie die Suche, um eine Adresse oder Zone zu finden.
  • 🟢 Klicken Sie auf die Zonen, um weitere Einzelheiten zu erfahren.
  • 📦 Finden Sie den nächstgelegenen Kompostplatz oder das nächstgelegene Büro, nachdem Sie einen Punkt ausgewählt haben.
  • 🧭 Zeichnen Sie eine Route vom Lager zu einem beliebigen Punkt, indem Sie die Adresse in der Spalte Zieladresse eingeben.
  • 🛣️ Route zwischen zwei verschiedenen Adressen.
  • 🔄 Wenn Sie die Suche löschen möchten, starten Sie die Seite neu
  • 🛠️ Wenn Sie etwas ändern oder verbessern möchten, wenden Sie sich an die technische Abteilung
